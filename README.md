## Crime Record & Law Enforcement Intelligence Database System (CRLEID)

Full-stack law enforcement database system built with Oracle SQL 21c, PL/SQL, Node.js, and HTML/CSS/JS.

### Features
- 9-table relational schema normalised to 3NF
- PL/SQL triggers for tamper-proof audit logging
- Recidivism risk scoring via scalar function fn_Recidivism_Score()
- Window functions: RANK, DENSE_RANK, LISTAGG, running SUM
- Node.js/Express REST API with browser dashboard
- 12/12 functional test cases passed

### Tech Stack
Oracle SQL 21c · PL/SQL · Node.js · Express · HTML · CSS · JavaScript
